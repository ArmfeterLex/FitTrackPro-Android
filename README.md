# FitTrack Pro Android

Мобильное приложение для отслеживания физической активности на Android, разработанное специально для российских пользователей.

## О проекте

FitTrack Pro решает проблему недостаточной физической активности через простой и элегантный подход. Приложение делает фитнес доступным и приятным для каждого.

## Функциональность

### 🏃 Базовый трекинг активности
- Подсчет шагов
- Расход калорий
- Пройденная дистанция

### ⏱️ Таймер тренировок
- С возможностью паузы и возобновления
- Идеально для интервальных занятий

### 🏆 Мотивационные челленджи
- "10 000 шагов в день"
- "Сжечь 500 калорий"
- Другие задания

### 📊 Статистика прогресса
- Детальная аналитика
- Визуализация достижений
- История тренировок

## Технические особенности

- **Платформа:** Android
- **Язык:** Java/Kotlin
- **Дизайн:** Material Design
- **База данных:** SQLite / Room
- **Минимальная версия:** Android 5.0 (API 21)

## Установка

1. Клонировать репозиторий
2. Открыть в Android Studio
3. Собрать проект (Build → Make Project)
4. Запустить на устройстве или эмуляторе

## Требования

- Android 5.0 (API 21) или выше
- Google Play Services (для подсчета шагов)
